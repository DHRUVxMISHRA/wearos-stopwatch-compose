# ⌚ Wear OS Stopwatch App

A modern **Stopwatch application for Wear OS** built using **Kotlin and Jetpack Compose for Wear OS**.
This app provides a clean circular UI optimized for smartwatches and demonstrates how to build responsive wearable applications using modern Android development tools.

---

## 🚀 Features

* ⏱ Start, Pause, and Reset stopwatch
* ⌚ UI optimized for **Wear OS circular displays**
* 🎨 Built with **Jetpack Compose for Wear OS**
* ⚡ Smooth animations and responsive layout
* 🔋 Lightweight and optimized for wearable devices

---

## 🛠 Tech Stack

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose for Wear OS
* **Platform:** Wear OS
* **IDE:** Android Studio

---

## 📱 Screens

* Stopwatch Main Screen
* Running Timer Screen
* Pause / Reset Controls
